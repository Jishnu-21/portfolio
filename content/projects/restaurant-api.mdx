Built a scalable RESTful API service that helps users discover restaurants based on their geographic location. The service utilizes MongoDB's geospatial queries for efficient location-based searches.

## Tech Stack
- Node.js & Express for the backend server
- MongoDB with geospatial indexing
- GeoJSON for location data
- JWT authentication
- Docker

## Key Features
- Location-based restaurant search
- Geospatial queries for nearby restaurants
- User authentication and authorization
- Restaurant data management
- Distance-based filtering
